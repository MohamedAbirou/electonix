### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/MohamedAbirou/e-shop.git
```


### Install packages

```shell
npm i
```

### Setup Prisma

```shell
npx prisma db push

```

### Start the app

```shell
npm run dev
```

## Available commands

Running commands with npm `npm run [command]`

| command         | description                              |
| :-------------- | :--------------------------------------- |
| `dev`           | Starts a development instance of the app |
